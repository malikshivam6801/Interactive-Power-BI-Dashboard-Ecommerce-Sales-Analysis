# Interactive-Power-BI-Dashboard-Ecommerce-Sales-Analysis

* Engineered a scalable Power BI dashboard to monitor and analyze nationwide online sales data.
* Applied advanced data modeling techniques including table relationships, data transformation, and DAX-based KPIs.
* Integrated user-driven parameters, slicers, and drill-down functionality for customizable analytics.
* Developed a wide range of visualizations (clustered bar, pie, donut, line, area, scatter, and geographic maps).
* Optimized dashboard usability and performance for enhanced business insights and decision-making.
